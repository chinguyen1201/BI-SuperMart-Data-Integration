# BI-SuperMart-Data-Integration
Build an ETL pipeline to integrate data from two separate sources — an MSSQL database and legacy Excel files — into a unified Dimensional Data Store (DDS). The goal is to consolidate product sales information across two supermarket branches.
- Link video demo: https://youtu.be/rdVDi1rPV6g
  
The project for the course **Information Systems for Business Intelligence - CSC12107 - HCMUS**
